# TrendNow

Spot rising topics before everyone else piles in.

TrendNow is a Chrome extension for Google Trends analysis, trending topic discovery, rising keyword research, and early trend tracking. It helps creators, marketers, researchers, YouTubers, and idea hunters find trending searches, breakout topics, and momentum signals before a niche gets crowded.

## Why People Download It

- Find breakout topics faster
- Scan trends by country or worldwide
- Search a keyword and surface rising related ideas
- Track hot signals from one compact popup
- Get notified when fresh topics appear

## What It Feels Like

TrendNow is built for the moment when you want to know:

- What is taking off right now?
- Is this trend still early?
- Which topic should I make content on next?
- What is heating up in another country before it gets crowded here?

## Install

1. Open `chrome://extensions/` in Chrome.
2. Enable `Developer mode`.
3. Click `Load unpacked`.
4. Select the extension folder you want to load.

For packaged final folders, use:

- `E:\saas\Finals\TrendNow-Public`
- `E:\saas\Finals\TrendNow-Admin`

## Use

1. Open the popup.
2. Choose a country or `Worldwide`.
3. Click `Scan Now`.
4. Search a keyword to find rising related topics.
5. Use:
   - `TRENDS` for live topics
   - `ALERTS` for hotter signals
   - `SIGNALS` for proof and scoring
   - `TRACK` for saved trends
   - `NOTIFY` for new-topic notifications
